# react-employ-app

## 项目介绍
- 具有实时聊天功能的招聘APP
- 前端(```React```, ```React-router-dom```, ```redux```, ```antd-mobild```, ```axios```)
- 后台(```MongoDB```, ```Socket.io```, ```Express```, ```Express```)
- 前后端连调采用 JWD 实现


### 安装依赖
```
npm install
npm install -g nodemon
```

### 启动项目 & 服务器
```
npm start

cd server & nodemon server.js
```

### 打包上线
```
npm run build
```
